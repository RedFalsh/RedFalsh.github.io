#screenkey

安装：
下载地址：https://launchpad.net/screenkey

下载screenkey-02.tar.gz文件包

解压：
`tar -zxvf screenkey-02.tar.gz`

进入目录执行：
`python2 setup.py install`

这里貌似只支持python2

需要安装pygtk软件包

`sudo python2 -m pip install pygtk`
